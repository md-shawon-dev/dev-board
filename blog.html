Question-1: Different Ways to Select an Element in the DOM

In JavaScript, there are multiple ways to select elements in the DOM.
The most common methods are:
document.getElementById("id"): Selects an element by its unique id.
document.getElementsByClassName("class"): Returns an HTMLCollection of elements matching the class name.
document.getElementsByTagName("tag"): Retrieves elements by their tag name, like div, p, etc.
document.querySelector("selector"): Selects the first element matching a CSS selector.
document.querySelectorAll("selector"): Returns a NodeList of all elements matching the CSS selector.
document.getElementsByName("name"): Selects elements by their name attribute, often used for form elements.

Question-1: Difference Between innerHTML, innerText, and textContent

innerHTML retrieves or sets the HTML content inside an element, meaning it includes HTML tags and formatting. If you assign a new value to innerHTML, it will parse any HTML within it. innerText retrieves only the visible text inside an element and ignores hidden elements or extra spaces. It also respects CSS styling like display: none. textContent retrieves all text within an element, including hidden elements, and preserves whitespace. Unlike innerHTML, it treats text as plain text and does not parse HTML.

Question-1: What is Event Delegation in the DOM?

Event delegation is a technique where you attach an event listener to a parent element instead of individual child elements. This is useful when handling events for dynamically added elements. It works because events bubble up from the child to the parent. Instead of adding a click listener to multiple buttons, you add a single listener to a common ancestor and use event.target to determine which child triggered the event.

Question-1: What is Event Bubbling in the DOM?

Event bubbling is a behavior in the DOM where an event triggered on a child element propagates (bubbles) up to its parent and ancestor elements. For example, if you click a button inside a div, the click event first fires on the button, then bubbles up to the div, then further up to body, html, and document.

Question-1: Creating, Adding, and Removing Elements in JavaScript

You can create, add, and remove elements dynamically using JavaScript.
Creating an element: Use document.createElement("tagName") to create a new element.
Adding an element: Use appendChild(), append(), or insertBefore() to insert the element into the DOM.
Removing an element: Use removeChild() or element.remove() to delete an element.
